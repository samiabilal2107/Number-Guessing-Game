🎯 Number Guessing Game
Java Console-Based Interactive Game 🎮

📌 Project Overview
This project is a Java-based Number Guessing Game developed using Object-Oriented Programming (OOP) concepts. The game generates a random number between 0 and 1000, and the player must guess the correct number within a limited number of attempts based on the selected difficulty level.

The game provides intelligent hints, tracks repeated guesses, and offers multiple difficulty modes to make the gameplay more challenging and engaging.

📖 Description

The Number Guessing Game challenges players to identify a randomly generated number through logical deduction and provided hints.

The game includes:
Three difficulty levels (Easy, Hard, Complex)
Dynamic hint system
Guess tracking mechanism
Duplicate guess detection
Remaining attempts display
Option to replay the game

Players receive feedback after every guess indicating whether their entered number is greater or smaller than the target number. Additional proximity hints help users narrow down the correct answer.

When only three attempts remain, players can request an extra hint revealing the quarter range in which the secret number lies.

🧠 Game Logic
Proximity Hints

The game provides hints based on how close the guess is to the actual number:

Difference ≤ 10 → "You are Very Close"
Difference ≤ 50 → "You are Close"
Quarter Range Hint

When only three guesses remain, the player can request a special hint:

0 – 250 → First Quarter
251 – 500 → Second Quarter
501 – 750 → Third Quarter
751 – 1000 → Fourth Quarter
