# 📌 Project Overview

This project is a **Java console-based Number Guessing Game** that challenges players to guess a randomly generated number between **0 and 1000**. It features multiple difficulty levels, intelligent hints, duplicate guess detection, and replay functionality to create an engaging gameplay experience.

---

# 📖 Description

The game generates a random number using Java's `Random` class and allows players to choose between **Easy**, **Hard**, and **Complex** difficulty levels. It provides directional hints, optional quarter-range hints, duplicate guess detection, and tracks all player guesses before allowing users to replay the game.
